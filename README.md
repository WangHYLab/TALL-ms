# TALL Paper Plotting Code Repository
This repository contains the complete plotting scripts and source code used to generate figures for the study:

**Bcl11a sustains DN2 thymocyte competition 1 to avert thymus autonomy and T-ALL**

The repository is intended to facilitate figure reproduction, data visualization, and methodological transparency associated with the publication.

## Repository Structure
```plaintext
├── data/       # Raw dataset files
├── script/     # All plotting source code
└── out/     # Generated figure results
```

## Usage
Modify the file paths in the scripts according to your local environment.
Run the code under the `script/` folder directly.
All output figures will be automatically saved to the `out/` directory.

## Notes
Please adjust the relative/absolute file paths in the code first to ensure the scripts run correctly on your machine.

If external datasets are required, corresponding download instructions or accession information should be included within the relevant scripts or publication.
